# Immigrant Community Connector

## Problem Statement
Many immigrants struggle to find people from their home country or city after relocating.
This can lead to social isolation and difficulty accessing culturally relevant support.

## Project Goal
The goal of this web application is to help immigrants connect with others from the same
country or city living nearby, fostering community, support, and shared resources.

## Technologies (Planned)
- Java
- Maven
- JSP / Servlets
- MySQL
- Tomcat
- GitHub
